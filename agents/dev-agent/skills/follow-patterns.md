# Skill: Follow Patterns

## When to invoke
Any code writing task. Always invoke alongside other dev skills to ensure consistency with project conventions.

## Steps
1. Read patterns.md in full before writing any code
2. Apply: naming conventions, file structure, import order, export style
3. Read guardrails.md — every forbidden pattern is a hard block
4. Mirror the structure of existing similar files in the project
5. If patterns.md is empty, derive conventions from existing code samples in spec

## Output format
No standalone output — this skill augments other skills.
