# Skill: Estimate Build Comparison

## When to invoke
Task involves comparing this build against what an agency or freelancer would charge.

## Steps
1. Size the built scope in features and screens from the actual output — not from the original wish list
2. Estimate agency cost using conservative market rates and a realistic timeline including discovery and revisions
3. Estimate freelancer cost the same way at a lower rate and longer calendar time
4. Put the session's actual dollar cost and same-day turnaround alongside — no editorializing, the numbers speak
5. State every assumption (rates used, scope counted); round agency/freelancer estimates down, never up

## Output format
// filepath: report/build-teardown.md (comparison section)
Three-column table (agency / freelancer / this session) with cost, timeline, assumptions list.
