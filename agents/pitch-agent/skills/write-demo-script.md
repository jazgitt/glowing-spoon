# Skill: Write Demo Script

## When to invoke
Task involves walking a prospect through the built MVP live or on a recorded video.

## Steps
1. Order the walkthrough by customer value, not by build order — lead with the feature that sells
2. For each step: what to click, what to say (one sentence), what the payoff on screen is
3. Only script features that exist in the built output; never demo vaporware
4. Include one recovery line for the most likely failure point ("if the email is slow, say...")
5. Target 3 minutes total — cut anything that doesn't move the sale

## Output format
// filepath: report/demo-script.md
Numbered steps: ACTION / SAY / PAYOFF. Three minutes maximum.
